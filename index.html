估計 · 連續 / 比例 DV
信賴區間 (CI)
Confidence Interval — 114-1 基礎統計
▼

信賴區間是用樣本統計量來估計母體參數可能落在的範圍，而非「母體參數有 95% 機率在區間內」。

解讀：若以相同方式重複抽樣 100 次，約有 95 次建構的 CI 會包含真正的母體參數 $\mu$。

⚠️ 錯誤解讀：「此區間有 95% 機率包含 $\mu$」—$\mu$ 是固定的，不是隨機的！

── 通用 CI 結構 ── $$\text{點估計} \pm \text{臨界值} \times \text{標準誤差} = \text{點估計} \pm \text{邊際誤差}$$ ── 影響 CI 寬窄的因素 ── 1. 信心水準 ↑ → CI 越寬（$z^*$ 或 $t^*$ 越大） 2. 樣本數 $n$ ↑ → CI 越窄（$SE = s/\sqrt{n}$ 越小） 3. 變異數 $s^2$ ↑ → CI 越寬
信心水準αz*（常態）適用時機
90%.101.645精度要求較低
95%.051.960最常用
99%.012.576要求嚴格
💡 σ 已知 → 用 z*（常態分布）；σ 未知（實務上幾乎都是）→ 用 t*(df=n−1)。樣本大（n≥30）時 t* ≈ z*。
── σ 已知（z CI）── $$CI = \bar{x} \pm z^*(\alpha/2) \times \frac{\sigma}{\sqrt{n}}$$ ── σ 未知（t CI，實務常用）── $$CI = \bar{x} \pm t^*\!\left(\alpha/2,\;df=n-1\right) \times \frac{s}{\sqrt{n}}$$ ── 標準誤 SE ── $$SE = \frac{s}{\sqrt{n}}$$
  1. 1
    確認資料型態
    連續 DV、σ 已知/未知 → 決定用 z 或 t
  2. 2
    查臨界值
    σ 未知：t*(df=n−1, α/2)；查 t 表找雙尾 α=.05 的值
  3. 3
    計算 SE
    SE = s / √n
  4. 4
    建構區間
    下界 = x̄ − t* × SE；上界 = x̄ + t* × SE
📂 講義例子 — 學測成績（gsat）

整體：ci means gsat, level(95) → 95% CI = [47.61, 48.04]

男生：CI = [47.07, 47.73]　女生：CI = [47.94, 48.51]

→ 兩個 CI 不重疊，表示男女學測平均有顯著差異（女高於男）

💡 CI 不重疊 ≈ 差異顯著，但這只是直觀判斷；正式推論仍要做假設檢定。
── 母體比例 CI ── $$CI = \hat{p} \pm z^*(\alpha/2) \times \sqrt{\frac{\hat{p}(1-\hat{p})}{n}}$$ ── 前提條件 ── $n \times \hat{p} \geq 10$ 且 $n \times (1-\hat{p}) \geq 10$（確保常態近似有效）
  1. 1
    計算樣本比例
    p̂ = x / n（x = 事件發生次數）
  2. 2
    計算 SE
    SE = √(p̂(1−p̂)/n)
  3. 3
    查 z*（固定用常態）
    95% CI → z* = 1.960
  4. 4
    建構區間
    下界 = p̂ − z*×SE；上界 = p̂ + z*×SE
📂 講義例子 — 私校就讀比例（priv）

指令：ci prop priv

若 p̂ = 0.35, n = 500 → SE = √(0.35×0.65/500) = 0.0213

95% CI = 0.35 ± 1.96 × 0.0213 = [0.308, 0.392]

⚠️ 比例 CI 一律用 z*（中央極限定理），不用 t*。
* ─── 平均數 CI（預設 95%）─── ci means gsat, level(95) * ─── 分組平均數 CI ─── ci means gsat, by(gender) level(95) * ─── 比例 CI ─── ci prop priv * ─── CI 圖（多個連續變數）─── ciplot chi eng math soc sci * ─── CI 圖（按分組）─── ciplot math, by(gender)
⚠️ 常見指令注意

• ci means：連續變數，可接 level(90/95/99)

• ci prop：0/1 二元變數，自動用 Wilson 法

• ciplot：視覺化多變數 CI，需先安裝（ssc install ciplot）

• by() 在 ci means 中需加 level() 才有 CI 輸出

假設檢定 · 單一母體
單母體假設檢定
z 檢定 / t 檢定 / 比例檢定 — 114-1
▼

用樣本資料檢定關於單一母體參數的假設（μ 或 p），推論母體是否等於某特定值。

三種假設方向（以平均數為例）：

雙尾：$H_0: \mu = \mu_0$ vs $H_1: \mu \neq \mu_0$

右尾：$H_0: \mu \leq \mu_0$ vs $H_1: \mu > \mu_0$

左尾：$H_0: \mu \geq \mu_0$ vs $H_1: \mu < \mu_0$

  1. 1
    設定假設（H₀ 和 H₁）
    確認方向（雙尾/左尾/右尾）、設定顯著水準 α（通常 .05）
  2. 2
    確認檢定類型
    σ 已知 → z 檢定；σ 未知（估計 s）→ t 檢定（df = n−1）
  3. 3
    計算檢定統計量
    z = (x̄ − μ₀) / (σ/√n)　或　t = (x̄ − μ₀) / (s/√n)
  4. 4
    找臨界值 / p 值
    查 z 表或 t 表（注意單尾/雙尾）；或用 Stata 得 p 值
  5. 5
    決策與結論
    p < α（或 |統計量| > 臨界值） → 拒絕 H₀；否則 → 保留 H₀
    結論需呼應研究問題，說明實質意義
Type I vs Type II 錯誤：
  • Type I (α)：H₀ 為真卻拒絕 → 假陽性，機率 = α（自己設定）
  • Type II (β)：H₁ 為真卻保留 H₀ → 假陰性
  • 檢定力 (Power) = 1 − β；n ↑ → power ↑

已知母體標準差 σ，或 n 足夠大（n≥30）時使用。

── z 統計量 ── $$z = \frac{\bar{x} - \mu_0}{\sigma / \sqrt{n}}$$ ── 決策 ── 雙尾：$|z| > z^*(\alpha/2)$ → 拒絕 $H_0$ 左尾：$z < -z^*(\alpha)$ → 拒絕 $H_0$ 右尾：$z > z^*(\alpha)$ → 拒絕 $H_0$
📂 講義例子 — 鑽頭直徑

H₀: μ = 32 mm　H₁: μ ≠ 32 mm（雙尾）

已知：σ = 4, n = 25, x̄ = 29.5, α = .01

計算：

SE = 4 / √25 = 4 / 5 = 0.8

z = (29.5 − 32) / 0.8 = −2.5 / 0.8 = −3.125

決策：|z| = 3.125 > z*(0.005) = 2.576 → 拒絕 H₀

p 值 = 2 × P(Z < −3.125) ≈ 0.0018 < .01 → 拒絕 H₀

結論：鑽頭平均直徑顯著偏離規格值 32 mm（z=−3.125, p=.0018）。

🖩 fx-82 — z 值計算

29.5−32= → −2.5
÷0.8= → −3.125

母體標準差 σ 未知，用樣本標準差 s 估計，df = n−1。

── t 統計量 ── $$t = \frac{\bar{x} - \mu_0}{s / \sqrt{n}}, \quad df = n - 1$$ ── 決策 ── $|t| > t^*(df,\;\alpha/2)$ → 拒絕 $H_0$（雙尾）
📂 講義例子 — 汽車修車費

H₀: μ = 200 元　H₁: μ > 200 元（右尾）

已知：n = 9, x̄ = 230, s = 48.35, α = .05（另外也測試 α=.01）

計算：

SE = 48.35 / √9 = 48.35 / 3 = 16.117

t = (230 − 200) / 16.117 = 30 / 16.117 = 1.861

df = 9 − 1 = 8

決策（右尾）：

t*(8, α=.05, 右尾) = 1.860 → t=1.861 > 1.860 → α=.05 拒絕 H₀

t*(8, α=.01, 右尾) = 2.896 → t=1.861 < 2.896 → α=.01 保留 H₀

結論：在 α=.05 時，修車費顯著高於 200 元；但在 α=.01 時，證據不足。

🖩 fx-82 — t 值計算

9√ → 3
48.35÷3= → 16.117（SE）
230−200= → 30
÷16.117= → 1.861

⚠️ 右尾 vs 雙尾：題目說「是否高於」→ 右尾，臨界值查單尾；「是否不同」→ 雙尾，查 α/2。
── 單母體比例 z 檢定 ── $H_0: p = p_0$ $$z = \frac{\hat{p} - p_0}{\sqrt{p_0(1-p_0)/n}}$$ ── 注意：SE 用 p₀ 而非 p̂（H₀ 假定 p₀ 為真）──
📂 例子 — 核能支持比例

H₀: p = 0.50（支持核能恰好 50%）　H₁: p ≠ 0.50

n = 400, p̂ = 0.44

SE = √(0.5×0.5/400) = √(0.000625) = 0.025

z = (0.44 − 0.50) / 0.025 = −0.06 / 0.025 = −2.40

|z|=2.40 > 1.96 → 拒絕 H₀，支持比例顯著不到 50%

💡 比例檢定的 SE 公式和 CI 不同！CI 用 p̂，假設檢定用 p₀。
* ─── z 檢定（σ 已知）─── ztest diameter == 32, sd(4) * ─── t 檢定（σ 未知，最常用）─── ttest repair_cost == 200 * ─── 比例 z 檢定 ─── prtest support_nuke == 0.5 * ─── 查 t 臨界值（invttail：右尾面積 p, df=8）─── di invttail(8, 0.05) // 右尾 α=.05 → 1.860 di invttail(8, 0.025) // 雙尾 α=.05 → 2.306 * ─── 查 z 臨界值 ─── di invnormal(0.975) // 雙尾 α=.05 → 1.96 di invnormal(0.995) // 雙尾 α=.01 → 2.576
⚠️ 注意事項

• ttest var == 值：單樣本 t（與常數比較）

• Stata 輸出同時給出左尾、右尾、雙尾 p 值，根據 H₁ 方向選正確那個

• invttail(df, p)：p 是右尾面積；雙尾 α=.05 → 用 p=0.025

假設檢定 · 雙母體
雙母體假設檢定
獨立 t / 相依 t / 比例差 — 114-1
▼

比較兩個群體的平均數或比例是否有顯著差異。關鍵：先判斷樣本是「獨立」還是「相依（配對）」。

　獨立樣本 t相依（配對）樣本 t
資料結構兩組不同人（如：男 vs 女）同一人測兩次或兩個配對變數
H₀μ₁ − μ₂ = 0μ_d = 0（差值平均 = 0）
前提先用 F 檢定確認變異數是否相等差值 d = v₁ − v₂ 服從常態
Statasdtest y,by(g) → ttest y,by(g)ttest v1 == v2
── 獨立樣本 t（等變異）── $$t = \frac{\bar{x}_1 - \bar{x}_2}{\sqrt{S_p^2\!\left(\frac{1}{n_1}+\frac{1}{n_2}\right)}}, \quad S_p^2 = \frac{(n_1-1)s_1^2+(n_2-1)s_2^2}{n_1+n_2-2}, \quad df = n_1+n_2-2$$ ── 相依樣本 t ── $$t = \frac{\bar{d}}{s_d/\sqrt{n}}, \quad df = n-1 \quad \text{（}d = v_1 - v_2\text{，每人差值）}$$
  1. 1
    先做變異數齊一性檢定（F 檢定）
    Stata：sdtest y, by(group)
    H₀: σ₁² = σ₂²；若 p > .05 → 等變異 → 用 ttest（合併）
    若 p < .05 → 不等變異 → 用 ttest y, by(g) unequal（Welch）
  2. 2
    設定假設
    H₀: μ₁ = μ₂（或 μ₁ − μ₂ = 0）；H₁: μ₁ ≠ μ₂
  3. 3
    計算 t 統計量
    等變異：合併方差 Sp²；Welch：Satterthwaite df（Stata 自動算）
  4. 4
    決策
    p < α → 拒絕 H₀，兩組平均數顯著不同
📂 講義例子 — 移民態度（all）by 大學別（uni）

H₀: 兩校移民態度平均數相等

Step 1：sdtest all, by(uni) → p 值確認變異數是否齊一

Step 2：若齊一 → ttest all, by(uni)；若不齊一 → ttest all, by(uni) unequal

另一例：trust_hlth vs trust_edu（相依樣本，見下頁）

💡 流程口訣：先 sdtest，再 ttest。不先做 F 檢定就直接 ttest 是最常見的扣分錯誤！

同一群人測量兩個相關變數（前測/後測、同一人的兩個態度），或自然配對的兩組。

── 相依樣本 t（Paired t-test）── $$d_i = v_{1i} - v_{2i} \quad \text{（每人差值）}, \quad \bar{d} = \frac{\sum d_i}{n}$$ $$t = \frac{\bar{d}}{s_d / \sqrt{n}}, \quad df = n - 1$$ ── H₀: μ_d = 0（兩變數平均相等）──
📂 講義例子 — 對醫療機構 vs 教育機構的信任度

H₀: μ(trust_hlth) = μ(trust_edu)　（即 μ_d = 0）

Stata：ttest trust_hlth == trust_edu

結果：t = 22.998，df 很大，p < .001

diff 的 95% CI = [0.517, 0.613]（不包含 0）

結論：台灣民眾對醫療機構的信任度顯著高於教育機構（t=22.998, p<.001），差異約 0.56 分。

⚠️ 配對 t 用 ttest v1 == v2，兩個變數在同一 row！獨立樣本用 ttest y, by(group)，不要搞混。
── 雙母體比例差 z 檢定 ── $H_0: p_1 = p_2$（或 $p_1 - p_2 = 0$） $$\hat{p}_c = \frac{x_1 + x_2}{n_1 + n_2} \quad \leftarrow \text{合併比例}$$ $$z = \frac{\hat{p}_1 - \hat{p}_2}{\sqrt{\hat{p}_c(1-\hat{p}_c)\!\left(\dfrac{1}{n_1}+\dfrac{1}{n_2}\right)}}$$
📂 講義例子 — 出國念書意願（exp）by 大學別（uni）

H₀: 兩校出國念書比例相等

Stata：prtest exp, by(uni)

→ Stata 輸出兩校各自 p̂、合併 p̂_c、z 值與 p 值

→ 若 p < .05 → 兩校出國意願比例顯著不同

💡 比例差用 z 檢定（不用 t），因為比例的抽樣分布用常態近似。前提：n₁p̂₁ ≥ 10, n₁(1−p̂₁) ≥ 10（兩組都要確認）。
* ─── Step 1：先做變異數齊一性 F 檢定 ─── sdtest all, by(uni) * ─── Step 2a：等變異 → 合併 t 檢定 ─── ttest all, by(uni) * ─── Step 2b：不等變異 → Welch t 檢定 ─── ttest all, by(uni) unequal * ─── 相依樣本（配對）t 檢定 ─── ttest trust_hlth == trust_edu * ─── 雙母體比例差 z 檢定 ─── prtest exp, by(uni)
⚠️ 常見錯誤整理

• 忘記 sdtest：直接做 ttest 而不先確認變異數是否相等

• 配對 vs 獨立搞混：同一人的兩個變數用 ttest v1==v2，不同人用 ttest y, by(g)

• prtest 的 by() 用法：prtest y, by(group)，y 必須是 0/1 二元變數

• Stata 的 ttest 輸出三個 p 值（Ha: diff<0, diff≠0, diff>0），根據 H₁ 方向選對應那個

ANOVA · 連續 DV
單因子 ANOVA
一個類別自變數 (k ≥ 3 組)
▼

比較 k 個（≥3）獨立組別的母體平均數是否相等，核心邏輯是「組間變異 vs 組內變異」。

$H_0: \mu_1 = \mu_2 = \cdots = \mu_k$（所有組平均數相等）

$H_1$：至少一組平均數不同（不需要指定哪組）

── 變異量分解 ── $$SST = SSB + SSW$$ ── 組間（Between Groups）── $$SSB = \sum_j n_j(\bar{x}_j - \bar{x})^2, \quad dfB = k-1, \quad MSB = \frac{SSB}{dfB}$$ ── 組內（Within Groups）── $$SSW = SST - SSB, \quad dfW = n-k, \quad MSW = \frac{SSW}{dfW}$$ ── F 統計量 ── $$F = \frac{MSB}{MSW} \sim F(dfB,\;dfW)$$ ── 效果量 ── $$\eta^2 = \frac{SSB}{SST} \quad \text{（0–1，越大效果越強）}$$
💡 判斷邏輯：若組間差異遠大於組內隨機誤差（F >> 1），則可拒絕 H₀。F ≥ F*(df₁, df₂, α) 則拒絕。
⚠️ 前提假設：各組獨立、各組來自常態分布、各組變異數相等（Bartlett test / Levene test）。違反變異數同質性時結果仍相對穩健（特別是樣本均等時）。
後驗比較（Post-hoc）：ANOVA 只告訴你「有差異」，但不知道哪兩組不同 → 需要 Tukey / Bonferroni / Scheffé。
  • Tukey：ω = q√(MSW/nⱼ)，各組 n 相等時最常用
  • Bonferroni：α' = α / C，C = 比較次數，最保守
  • Scheffé：最嚴格，適合複雜比較
  1. 1
    設定假設
    H₀: μ₁=μ₂=…=μₖ；H₁: 至少一對不等；設定 α（通常 .05）
  2. 2
    計算各組統計量
    各組樣本數 nⱼ、平均數 x̄ⱼ、全體平均 x̄ = Σnⱼx̄ⱼ / n
  3. 3
    計算 SSB（組間變異）
    SSB = Σⱼ nⱼ(x̄ⱼ − x̄)²　→　dfB = k−1　→　MSB = SSB/dfB
  4. 4
    計算 SSW（組內變異）
    若題目給 SST：SSW = SST − SSB；否則用各組方差累加
    →　dfW = n−k　→　MSW = SSW/dfW
  5. 5
    計算 F 統計量
    F = MSB / MSW，查 F 分布表找臨界值 F*(dfB, dfW, α)
  6. 6
    決策
    若 F ≥ F* → 拒絕 H₀；否則保留 H₀。引用 p 值更佳：p < α 則拒絕
  7. 7
    效果量
    η² = SSB / SST（小=.01，中=.06，大=.14）
  8. 8
    後驗比較（若顯著）

ANOVA · 連續 DV
二因子 ANOVA
兩個類別自變數 + 交互作用
▼

同時檢驗 A、B 兩個自變數對 Y 的效果，以及它們的交互作用（A×B）。

$H_{0A}$：A 無主效果（各 A 組平均相等）

$H_{0B}$：B 無主效果（各 B 組平均相等）

$H_{0AB}$：A×B 無交互作用（兩條線平行）

三組假設各自獨立，需分別檢定，各自有 F 值

── 變異量分解 ── $$SST = SS_A + SS_B + SS_{AB} + SS_{error}$$ ── 自由度 ── $$df_A = a-1 \;\text{（}a=\text{A 的水準數）}, \quad df_B = b-1 \;\text{（}b=\text{B 的水準數）}$$ $$df_{AB} = (a-1)(b-1), \quad df_{err} = ab(n-1), \quad df_T = N-1 = abn-1$$ ── F 統計量（三個都除以同一個 MS_error）── $$F_A = \frac{MS_A}{MS_{error}}, \quad F_B = \frac{MS_B}{MS_{error}}, \quad F_{AB} = \frac{MS_{AB}}{MS_{error}}$$
⚠️ 先看交互作用！若 F_AB 顯著，A、B 的主效果解釋需謹慎，應畫交互作用圖（Interaction Plot）來解釋。
交互作用的意思：A 對 Y 的影響因 B 的不同而不同（反之亦然）。圖上兩條線若非平行即有交互作用。

符號定義：a=A水準數、b=B水準數、n=每格樣本數、N=abn；Tᵢⱼ=格子加總、Tᵢ.=A第i列加總、T.ⱼ=B第j行加總、T..=全體加總

  1. 1
    設定三組假設（主效果 A、主效果 B、交互 A×B），設 α=.05
  2. 2
    整理資料：計算各格加總 Tᵢⱼ 與各邊際加總
    $$T_{i\cdot} = \sum_j T_{ij} \;\text{（A 第 }i\text{ 水準列加總）}, \quad T_{\cdot j} = \sum_i T_{ij} \;\text{（B 第 }j\text{ 水準行加總）}, \quad T_{\cdot\cdot} = \sum_i\sum_j T_{ij}$$
  3. 3
    計算校正因子 CF（Correction Factor）
    $$CF = \frac{T_{\cdot\cdot}^2}{N}$$
    CF 是所有後續 SS 計算的基準，先算好備用
  4. 4
    計算 SST（全體變異）
    $$SST = \sum_i\sum_j\sum_k x_{ijk}^2 - CF$$
    把每一個原始觀測值平方後加總，再減 CF
  5. 5
    計算 SS_A（因子 A 的主效果變異）
    $$SS_A = \frac{\displaystyle\sum_i T_{i\cdot}^2}{bn} - CF$$
    用 A 各水準的列加總平方後除以「每列格數×每格n」，再減 CF
  6. 6
    計算 SS_B（因子 B 的主效果變異）
    $$SS_B = \frac{\displaystyle\sum_j T_{\cdot j}^2}{an} - CF$$
    用 B 各水準的行加總平方後除以「每行格數×每格n」，再減 CF
  7. 7
    計算 SS_cells（各格細格變異）
    $$SS_{cells} = \frac{\displaystyle\sum_i\sum_j T_{ij}^2}{n} - CF$$
    用每個格子加總平方後除以 n，加總後減 CF
  8. 8
    計算 SS_AB（交互作用變異）← 關鍵！
    $$SS_{AB} = SS_{cells} - SS_A - SS_B$$
    細格變異扣掉兩個主效果，剩下的就是交互作用
  9. 9
    計算 SS_error（誤差）
    $$SS_{error} = SST - SS_{cells}$$
    也可用：SST − SS_A − SS_B − SS_AB
  10. 10
    計算 MS 與 F
    $$MS_A = \frac{SS_A}{df_A},\quad MS_B = \frac{SS_B}{df_B},\quad MS_{AB} = \frac{SS_{AB}}{df_{AB}},\quad MS_{err} = \frac{SS_{err}}{df_{err}}$$ $$F_A = \frac{MS_A}{MS_{err}},\quad F_B = \frac{MS_B}{MS_{err}},\quad F_{AB} = \frac{MS_{AB}}{MS_{err}}$$
  11. 11
    決策順序：先交互，後主效果
    ① 若 F_AB 顯著 → 畫互動圖，做 Simple Effects 分析，主效果解釋需謹慎
    ② 若 F_AB 不顯著 → 再解讀 F_A 與 F_B 的主效果

卡方檢定 · 類別 DV
χ² 適合度檢定
Goodness-of-Fit Test
▼

一個類別變數的觀察次數分布是否符合某個理論（預期）比例？

$H_0: p_1=p_{10},\;p_2=p_{20},\;\ldots,\;p_k=p_{k0}$（觀察分布符合理論）

$H_1$：至少一個 $p_i \neq p_{i0}$（不符合理論分布）

── 計算期望次數 ── $$E_i = n \times p_{i0} \quad \text{（}n\text{ 為總樣本數，}p_{i0}\text{ 為理論比例）}$$ ── 卡方統計量 ── $$\chi^2 = \sum_i \frac{(O_i - E_i)^2}{E_i}$$ ── 自由度 ── $$df = k - 1 \quad \text{（}k\text{ 為類別數）}$$ ── 決策 ── $\chi^2 \geq \chi^{2*}(df,\;\alpha)$ → 拒絕 $H_0$
💡 前提：每個類別的期望次數 Eᵢ ≥ 5（若違反需合併類別或用 Fisher exact test）
⚠️ 適合度檢定只有一個變數，不分 row/column。若題目有兩個變數 → 用獨立性/同質性檢定！
  1. 1
    設定假設
    H₀: p₁=p₁₀, …；H₁: 至少一個不符
  2. 2
    計算期望次數
    Eᵢ = n × pᵢ₀　確認所有 Eᵢ ≥ 5
  3. 3
    建立計算表格
    列出 Oᵢ, Eᵢ, (Oᵢ−Eᵢ), (Oᵢ−Eᵢ)², (Oᵢ−Eᵢ)²/Eᵢ
  4. 4
    計算 χ²
    加總所有 (O−E)²/E
  5. 5
    自由度
    df = k − 1
  6. 6
    決策與結論
    χ² ≥ χ²* → 拒絕 H₀；寫出具體結論
⚠️ 若理論比例是「均等」（每組相等），則 Eᵢ = n/k

• |SR| ≤ 2：該格子符合獨立假設，無顯著偏離

迴歸分析 (OLS)
量化解釋變項對結果變項的邊際效應
▼

迴歸分析的核心任務是將「相關性」轉化為具備解釋力的預測模型，量化 X 對 Y 的具體邊際影響，而非單純描述共同變動。

$H_0: \beta = 0$（X 對 Y 無線性影響）

$H_1: \beta \neq 0$（X 對 Y 有顯著影響）

── OLS 模型 ── $$Y = \beta_0 + \beta_1 X_1 + \beta_2 X_2 + \cdots + \varepsilon$$ ── 誤差項（核心精髓）── $$\varepsilon \sim i.i.d.\; N(0,\;\sigma^2)$$ ── OLS 估計準則 ── 最小化 $\displaystyle\sum_i (Y_i - \hat{Y}_i)^2$（殘差平方和） ── 變異量分解 ── $$SST = SSR + SSE, \quad R^2 = \frac{SSR}{SST}$$
💡 隨機關係 vs 確定關係：社會科學探討「隨機關係（Stochastic）」，誤差項 ε 承載了個人能力、運氣與所有未觀測的隨機變異，是模型的靈魂所在。
⚠️ 多元迴歸的「其他條件不變」：β₁ 代表在控制所有其他 X 的情況下，X₁ 增加 1 單位對 Y 的淨效果（邊際效應）。
① 模型擬合度：R² 與 Adjusted R²
── 判定係數 ── $$R^2 = \frac{SSR}{SST} \quad \rightarrow \quad \text{Y 的總變異中，模型能解釋的比例}$$ ── 調整後判定係數（防過度擬合）── $$\text{Adj. }R^2 = 1 - (1-R^2)\times\frac{n-1}{n-k-1}$$ → 加入無意義變項時，Adj. $R^2$ 不升反降 → 比較不同複雜度模型時的標準工具
💡 策略意義：若新增一個 X 後 Adj. R² 未提升，代表該變項無邊際貢獻，應保持模型簡潔。
② 統計顯著性：p-value

p-value 代表在 H₀（β=0）為真的前提下，觀察到目前樣本效應的機率。p < .05 僅是「效應存在」的最低門檻，不代表效應的實質重要性。

⚠️ 顯著性受樣本數影響極大——大樣本中，微小差異也可能 p < .05，必須結合係數大小進行判斷。
③ 係數解釋：β 的邊際效應
── 原始係數 (Unstandardized) ── $\beta_1 = 5$ → $X$ 每增加 1 單位，$Y$ 預期增加 5 單位（以原始測量單位解讀） ── 標準化係數 (Standardized, Beta) ── → 將所有變項轉為 $Z$ 分數後計算 → 消除測量單位差異，可直接比較不同 $X$ 對 $Y$ 的相對貢獻強度
💡 比較「教育年數」vs「工作經驗」對薪資的影響強度 → 必須用標準化係數（reg y x1 x2, beta）

OLS 估計量要成為「最佳線性不偏估計量（BLUE）」，必須符合 L.I.N.E. 四大假設。違反任一項都會動搖統計推論的有效性。

⚠️ 常見指令錯誤

• vif、hettest、rvfplot 都是後驗指令，必須在 reg 執行後才能使用

• avplot 後接的是自變項名稱，不是依變項

• beta 選項只影響輸出，不改變模型本身

統計指標只是工具，真正的洞察在於研究者如何結合統計數據與社會科學理論，區分「看到的現象」與「背後的真相」。

相關 vs 迴歸 vs 因果
Pearson r：只呈現共同變化趨勢，不區分因果方向
OLS 迴歸：量化影響力，但無法自動排除虛假相關
⚠️ 虛假相關（Spurious Association）範例：
教育（X）與薪資（Y）高度相關，但若忽略「能力（Z）」這一共同影響因子——Z 同時驅動 X 與 Y——我們觀察到的可能只是 Z 的效果，而非 X 對 Y 的真實因果。
因果階梯問題類型代表語句
關聯
P(Y|X)		 觀察連動規律 「如果我觀察到 X，Y 會怎樣？」
介入
P(Y|do(X))		 預測刻意介入的效果 「如果我做了 X，Y 會如何改變？」
反事實
Counterfactual		 想像未發生的情況 「如果 X 沒有發生，Y 會如何不同？」
💡 核心結語：統計顯著性（p < .05）絕不等於因果關係。唯有通過嚴謹的 L.I.N.E. 診斷確保模型有效，並輔以理論架構排除非因果影響，才能在數據中捕捉真實的社會邏輯。
