ANOVA · 連續 DV
單因子 ANOVA
一個類別自變數 (k ≥ 3 組)
▼

比較 k 個（≥3）獨立組別的母體平均數是否相等，核心邏輯是「組間變異 vs 組內變異」。

H₀：μ₁ = μ₂ = … = μₖ　（所有組平均數相等）

H₁：至少一組平均數不同（不需要指定哪組）

── 變異量分解 ── SST = SSB + SSW ── 組間（Between Groups）── SSB = Σⱼ nⱼ (x̄ⱼ − x̄)² dfB = k − 1 MSB = SSB / dfB ── 組內（Within Groups）── SSW = SST − SSB dfW = n − k MSW = SSW / dfW ── F 統計量 ── F = MSB / MSW ← F ~ F(dfB, dfW) ── 效果量 ── η² = SSB / SST (0–1，越大效果越強)
💡 判斷邏輯：若組間差異遠大於組內隨機誤差（F >> 1），則可拒絕 H₀。F ≥ F*(df₁, df₂, α) 則拒絕。
⚠️ 前提假設：各組獨立、各組來自常態分布、各組變異數相等（Bartlett test / Levene test）。違反變異數同質性時結果仍相對穩健（特別是樣本均等時）。
後驗比較（Post-hoc）：ANOVA 只告訴你「有差異」，但不知道哪兩組不同 → 需要 Tukey / Bonferroni / Scheffé。
  • Tukey：ω = q√(MSW/nⱼ)，各組 n 相等時最常用
  • Bonferroni：α' = α / C，C = 比較次數，最保守
  • Scheffé：最嚴格，適合複雜比較
  1. 1
    設定假設
    H₀: μ₁=μ₂=…=μₖ；H₁: 至少一對不等；設定 α（通常 .05）
  2. 2
    計算各組統計量
    各組樣本數 nⱼ、平均數 x̄ⱼ、全體平均 x̄ = Σnⱼx̄ⱼ / n
  3. 3
    計算 SSB（組間變異）
    SSB = Σⱼ nⱼ(x̄ⱼ − x̄)²　→　dfB = k−1　→　MSB = SSB/dfB
  4. 4
    計算 SSW（組內變異）
    若題目給 SST：SSW = SST − SSB；否則用各組方差累加
    →　dfW = n−k　→　MSW = SSW/dfW
  5. 5
    計算 F 統計量
    F = MSB / MSW，查 F 分布表找臨界值 F*(dfB, dfW, α)
  6. 6
    決策
    若 F ≥ F* → 拒絕 H₀；否則保留 H₀。引用 p 值更佳：p < α 則拒絕
  7. 7
    效果量
    η² = SSB / SST（小=.01，中=.06，大=.14）
  8. 8
    後驗比較（若顯著）
    Tukey：先計算 ω = q × √(MSW/n)，|x̄ᵢ − x̄ⱼ| > ω 則兩組顯著不同
💡 考試填表格時，先把 df 填好，再回頭算 SS，最後算 MS 和 F，不容易搞混。

ANOVA · 連續 DV
二因子 ANOVA
兩個類別自變數 + 交互作用
▼

同時檢驗 A、B 兩個自變數對 Y 的效果，以及它們的交互作用（A×B）。

H₀ₐ：A 無主效果（各 A 組平均相等）

H₀ᵦ：B 無主效果（各 B 組平均相等）

H₀ₐᵦ：A×B 無交互作用（兩條線平行）

三組假設各自獨立，需分別檢定，各自有 F 值

── 變異量分解 ── SST = SS_A + SS_B + SS_AB + SS_error ── 自由度 ── df_A = a − 1 ← a = A 的水準數 df_B = b − 1 ← b = B 的水準數 df_AB = (a−1)(b−1) df_err = ab(n−1) ← n = 每細格樣本數 df_T = N − 1 = abn − 1 ── F 統計量（三個都除以同一個 MS_error）── F_A = MS_A / MS_error F_B = MS_B / MS_error F_AB = MS_AB / MS_error
⚠️ 先看交互作用！若 F_AB 顯著，A、B 的主效果解釋需謹慎，應畫交互作用圖（Interaction Plot）來解釋。
交互作用的意思：A 對 Y 的影響因 B 的不同而不同（反之亦然）。圖上兩條線若非平行即有交互作用。

符號定義：a=A水準數、b=B水準數、n=每格樣本數、N=abn；Tᵢⱼ=格子加總、Tᵢ.=A第i列加總、T.ⱼ=B第j行加總、T..=全體加總

  1. 1
    設定三組假設（主效果 A、主效果 B、交互 A×B），設 α=.05
  2. 2
    整理資料：計算各格加總 Tᵢⱼ 與各邊際加總
    Tᵢ. = Σⱼ Tᵢⱼ ← 第 i 列（A 第 i 水準）的所有格加總 T.ⱼ = Σᵢ Tᵢⱼ ← 第 j 行（B 第 j 水準）的所有格加總 T.. = Σᵢ Σⱼ Tᵢⱼ ← 全體總加總
  3. 3
    計算校正因子 CF（Correction Factor）
    CF = T².. / N
    CF 是所有後續 SS 計算的基準，先算好備用
  4. 4
    計算 SST（全體變異）
    SST = Σᵢ Σⱼ Σₖ x²ᵢⱼₖ − CF
    把每一個原始觀測值平方後加總，再減 CF
  5. 5
    計算 SS_A（因子 A 的主效果變異）
    SS_A = Σᵢ Tᵢ.² / (bn) − CF
    用 A 各水準的列加總平方後除以「每列格數×每格n」，再減 CF
  6. 6
    計算 SS_B（因子 B 的主效果變異）
    SS_B = Σⱼ T.ⱼ² / (an) − CF
    用 B 各水準的行加總平方後除以「每行格數×每格n」，再減 CF
  7. 7
    計算 SS_cells（各格細格變異）
    SS_cells = Σᵢ Σⱼ Tᵢⱼ² / n − CF
    用每個格子加總平方後除以 n，加總後減 CF
  8. 8
    計算 SS_AB（交互作用變異）← 關鍵！
    SS_AB = SS_cells − SS_A − SS_B
    細格變異扣掉兩個主效果，剩下的就是交互作用
  9. 9
    計算 SS_error（誤差）
    SS_error = SST − SS_cells
    也可用：SST − SS_A − SS_B − SS_AB
  10. 10
    計算 MS 與 F
    MS_A = SS_A / df_A MS_B = SS_B / df_B MS_AB = SS_AB / df_AB MS_err = SS_err / df_err F_A = MS_A / MS_err F_B = MS_B / MS_err F_AB = MS_AB / MS_err
  11. 11
    決策順序：先交互，後主效果
    ① 若 F_AB 顯著 → 畫互動圖，做 Simple Effects 分析，主效果解釋需謹慎
    ② 若 F_AB 不顯著 → 再解讀 F_A 與 F_B 的主效果

卡方檢定 · 類別 DV
χ² 適合度檢定
Goodness-of-Fit Test
▼

一個類別變數的觀察次數分布是否符合某個理論（預期）比例？

H₀：p₁=p₁₀, p₂=p₂₀, …, pₖ=pₖ₀　（觀察分布符合理論）

H₁：至少一個 pᵢ ≠ pᵢ₀　（不符合理論分布）

── 計算期望次數 ── Eᵢ = n × pᵢ₀ ← n 為總樣本數，pᵢ₀ 為理論比例 ── 卡方統計量 ── χ² = Σᵢ (Oᵢ − Eᵢ)² / Eᵢ ── 自由度 ── df = k − 1 ← k 為類別數 ── 決策 ── χ² ≥ χ²*(df, α) → 拒絕 H₀
💡 前提：每個類別的期望次數 Eᵢ ≥ 5（若違反需合併類別或用 Fisher exact test）
⚠️ 適合度檢定只有一個變數，不分 row/column。若題目有兩個變數 → 用獨立性/同質性檢定！
  1. 1
    設定假設
    H₀: p₁=p₁₀, …；H₁: 至少一個不符
  2. 2
    計算期望次數
    Eᵢ = n × pᵢ₀　確認所有 Eᵢ ≥ 5
  3. 3
    建立計算表格
    列出 Oᵢ, Eᵢ, (Oᵢ−Eᵢ), (Oᵢ−Eᵢ)², (Oᵢ−Eᵢ)²/Eᵢ
  4. 4
    計算 χ²
    加總所有 (O−E)²/E
  5. 5
    自由度
    df = k − 1
  6. 6
    決策與結論
    χ² ≥ χ²* → 拒絕 H₀；寫出具體結論
⚠️ 若理論比例是「均等」（每組相等），則 Eᵢ = n/k

χ² 獨立性／同質性檢定
Independence / Homogeneity Test
▼
　獨立性檢定同質性檢定
取樣方式單一樣本，測量兩個變數多個獨立樣本，各測一個變數

H₀：兩個類別變數互相獨立（無關聯）

H₁：兩個類別變數有關聯

── 期望次數（關鍵公式！）── Eᵢⱼ = (Rᵢ × Cⱼ) / n = 第 i 列總和 × 第 j 行總和 / 總 n ── 卡方統計量 ── χ² = Σᵢ Σⱼ (Oᵢⱼ − Eᵢⱼ)² / Eᵢⱼ ── 自由度 ── df = (R−1)(C−1) ← R=列數, C=行數 ── 標準化殘差 ── SR_ij = (Oᵢⱼ − Eᵢⱼ) / √Eᵢⱼ |SR| > 2 → 該格顯著偏離期望
💡 效果量：Cramér's V = √(χ²/(n×min(R−1,C−1)))，0=無關，1=完全相關
⚠️ 前提：所有格 Eᵢⱼ ≥ 5（至少 80% 的格）；若違反需合併類別
  1. 1
    設定假設
    H₀: 兩變數獨立（無關聯）；H₁: 有關聯；設 α=.05
  2. 2
    建立列聯表（觀察次數）
    確認行列加總正確；計算總 n
  3. 3
    計算所有格的期望次數
    Eᵢⱼ = (列總和 × 行總和) / n；確認 Eᵢⱼ ≥ 5
  4. 4
    計算 χ²
    對每一格計算 (O−E)²/E，然後加總
  5. 5
    自由度與臨界值
    df = (R−1)(C−1)；查 χ²*(df, α=.05)
  6. 6
    決策與結論
    χ² ≥ χ²* → 拒絕 H₀，結論要有實質意義
  7. 7
    標準化殘差（若顯著）
    計算 SR = (O−E)/√E，找出哪些格子顯著偏離
    |SR| > 2 → 該格子有顯著差異
📂 實習作業 1 — mom × polview（4×5 列聯表）

研究問題：媽媽教育程度（mom，4類）與政治觀點（polview，5類）是否有關聯？

df = (4−1)(5−1) = 3 × 4 = 12（可能因合併而調整）

χ² = 49.66

χ²*(12, .05) = 21.03 → 49.66 >> 21.03 → 拒絕 H₀

結論：媽媽教育程度與政治觀點有顯著關聯（χ²=49.66, df=12, p<.001）。透過標準化殘差可找出哪些組合特別突出。

📂 正課作業 2 — 政治保守主義跨國同質性（2×5）

研究問題：各國民眾的政治保守主義分布是否相同？（同質性檢定）

df = (2−1)(5−1) = 1 × 4 = 4

5個國家，每國分保守/不保守

雖然研究設計是「多個獨立樣本（各國）」，但計算方式與獨立性檢定完全相同！

💡 標準化殘差解讀（重要！）

SR = (O−E)/√E

• SR > +2：該格子的觀察次數顯著多於期望（正向偏離）

• SR < −2：該格子的觀察次數顯著少於期望（負向偏離）

• |SR| ≤ 2：該格子符合獨立假設，無顯著偏離

* ─── 基本獨立性檢定 ─── tabulate mom polview, chi2 * ─── 含期望次數 ─── tabulate mom polview, chi2 expected * ─── 標準化殘差（cchi2 = cell chi-squared contribution）─── tabulate mom polview, chi2 expected cchi2 * ─── 含效果量 Cramér's V ─── tabulate mom polview, chi2 expected V * ─── 含列%、行% ─── tabulate mom polview, chi2 row col * ─── 完整輸出 ─── tabulate mom polview, chi2 expected V cchi2 row col

• 第一個變數 = 列（row）；第二個 = 行（column）

• cchi2 輸出的是每格對 χ² 的貢獻，需自行取符號開根號才是標準化殘差

• 若類別太多導致 Eᵢⱼ < 5，Stata 會在輸出中警告
